# Simple Katanas

A Fabric mod that adds timing-based katana combat to Minecraft, featuring parries, perfect parry windows, visual feedback, custom sounds, and balanced, skill-focused gameplay.

## Features
- Timing-based parry & perfect parry
- Visual parry window HUD
- Custom sounds & particles
- Tiered katanas (wood, stone, metal)
- Balanced, skill-focused combat

## Requirements
- Minecraft 1.21.1
- Fabric Loader
- Fabric API

## Installation
Drop the mod `.jar` into your `mods` folder.
